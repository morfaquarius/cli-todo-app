# Todo App (CLI)

Консольное мини-приложение для управления списком задач.
Проект выполнен в рамках курса **«Go-разработчик с нуля»**.

Приложение позволяет добавлять задачи, просматривать список, отмечать задачи выполненными, удалять их, а также импортировать и экспортировать задачи в форматах **JSON** и **CSV**.

## Возможности

- ✅ **Добавление задач** - быстрое добавление новых задач со своим описанием
- ✅ **Просмотр задач** - вывод списка задач с фильтрацией по статусу
- ✅ **Отметить выполненной** - изменение статуса задачи на выполненную
- ✅ **Удаление задач** - удаление задач из списка
- ✅ **Экспорт данных** - сохранение задач в формате JSON или CSV
- ✅ **Импорт данных** - загрузка задач из файла

## Требования

- Go 1.16 или выше

## Установка

Клонируйте репозиторий и перейдите в директорию проекта:

```bash
cd todo-app
```

## Сборка

```bash
go build -o todo.exe ./cmd/todo
```

## Использование

### Добавление задачи

```bash
./todo.exe add -desc "Моя первая задача"
```

### Просмотр задач

Все задачи:
```bash
./todo.exe list
```

Только выполненные:
```bash
./todo.exe list -filter done
```

Только невыполненные:
```bash
./todo.exe list -filter pending
```

### Отметить задачу выполненной

```bash
./todo.exe complete -id 1
```

### Удалить задачу

```bash
./todo.exe delete -id 1
```

### Экспорт задач

В формате CSV:
```bash
./todo.exe export -format csv -out backup.csv
```

В формате JSON:
```bash
./todo.exe export -format json -out backup.json
```

### Импорт задач

```bash
./todo.exe load -file backup.json
```

## Формат данных

Задачи сохраняются в файле `tasks.json` по умолчанию в следующем формате:

```json
[
  {
    "id": 1,
    "description": "Моя первая задача",
    "done": false
  },
  {
    "id": 2,
    "description": "Вторая задача",
    "done": true
  }
]
```
